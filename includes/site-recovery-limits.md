以下限制适用于 Azure 站点恢复：

| 限制标识符 | 默认限制 |
| --- | --- | --- | --- |
| 每个订阅的保管库数 |500 |     
| 每个 Azure 保管库的服务器数 |250 |
| 每个 Azure 保管库的保护组数 |无限制 |
| 每个 Azure 保管库的恢复计划数 |无限制 |
| 每个保护组的服务器数 |无限制 |
| 每个恢复计划的服务器数 |50 |

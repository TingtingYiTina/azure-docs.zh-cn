# <a name="discover-and-share-collections-in-azure-ai-gallery"></a>在 Azure AI 库中发现和共享收藏

可使用 [Azure AI 库](http://gallery.cortanaintelligence.com)中的[收藏](https://gallery.cortanaintelligence.com/collections)将用于特定解决方案或概念的试验、API 以及其他库项组合在一起。 可以将它们组合在一起，以便使用、保存以供日后参考或者共享。

## <a name="discover"></a>发现
若要浏览[库中](http://gallery.cortanaintelligence.com)的收藏，请在“更多”下选择“收藏”。

![在库主页上选择“收藏”](./media/gallery-collections/select-collections-in-gallery.png)

“[收藏](https://gallery.cortanaintelligence.com/collections)”页显示最近添加的收藏和热门收藏列表。 若要查看所有收藏，请选择“查看全部”。 要搜索特定收藏，请选择“查看全部”，并选择筛选条件。 也可以在库页面顶部的“搜索”框中输入搜索项。

可在收藏详细信息页上获取有关收藏的详细信息。 若要打开某个收藏详细信息页，请选择收藏。 在收藏详细信息页的“评论”部分中，可对收藏进行评论、提供反馈或提问。 甚至可以在 Twitter 或 LinkedIn 上与好友或同事分享收藏。 还可以通过电子邮件发送指向收藏详细信息页的链接，来邀请其他用户查看该页面。

![与朋友共享此项](./media/gallery-how-to-use-contribute-publish/share-links.png)

![添加自己的备注](./media/gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>参与
可以创建包含自己所拥有的项或由社区提供的项的收藏。 收藏中的项可以是提出特定解决方案或概念的任意库项。

例如，可以使用收藏将有关特定主题的项组合在一起，还可以将解决复杂问题的多步骤实验组合在一起。 由 Microsoft 提供的初始收藏包含多步骤机器学习实验模板。 可使用这些模板来解决实际问题，如联机欺诈检测、文本分类、零售预测或预测维护。

### <a name="create-a-collection"></a>创建集合

1. 使用 Microsoft 帐户登录到库。

2.  选择页面顶部的图像，并选择你的姓名。
  
    ![选择帐户名](./media/gallery-collections/click-account-name.png)

3. 选择“新建项”。
   
    ![选择“新建项”](./media/gallery-collections/click-new-item.png)
4. 在“说明”页上，为“项类型”选择“收藏”。 输入收藏的名称、简短摘要、详细说明和任何可能有助于其他用户找到收藏的标记。
   
    ![输入新收藏的信息](./media/gallery-collections/create-collection-page-1.png)
5. 在“图像选择”页上，选择随收藏一同显示的图像。 可以上传自己的图像文件，或者选择内置图像。 选择可帮助用户标识收藏内容和用途的图像。
   
    ![输入新收藏的信息](./media/gallery-collections/create-collection-page-2.png)
6. 在“设置”页上，针对“可见性”选择收藏是“公开”（任何人都可以查看收藏）还是“未列出”（仅具有指向收藏的直接链接的人员才可以查看它）。
   
   > [!IMPORTANT]
   > 将收藏设置为“公开”并选择“创建”后，以后不能将收藏更改为“未列出”。
   > 
   > 
   
    ![选择“公开”或“未列出”](./media/gallery-collections/create-collection-page-3.png)
7. 选择“创建”。

现在收藏已在 Azure AI 库中。 收藏会在帐户页的“收藏”选项卡上列出。

### <a name="add-items-to-a-collection"></a>向收藏添加项
要将项添加到集合，则打开收藏，然后依次选择“编辑”和“添加项”。

![通过详细信息页向收藏添加项](./media/gallery-collections/add-to-collection-from-details-page.png)

已分配到库的项列表会显示在帐户页上。 还可以在库中搜索要添加的项。 选择要添加的项。 所选的每个项都包含在要添加的项集内。 “添加”按钮指示已选择的项数目。

![添加到收藏](./media/gallery-collections/add-to-collection.png)

如果在浏览库时找到要添加的项，选择“添加到收藏”可打开项。 选择要向其中添加项的收藏。

![通过详细信息页向收藏添加项](./media/gallery-collections/add-to-collection-from-item-details.png)

要更改收藏的摘要、说明或标记，请打开收藏，并选择“编辑”。 

编辑收藏时，可以更改项在收藏中的顺序。 使用项旁边的箭头按钮，可在顺序列表中向上或向下移动它。 

要向收藏中的项添加备注，则选择项的右上角，并选择“添加/编辑备注”。 

若要从收藏中移除一个项，请选择“**移除**”。

![更改收藏的详细信息](./media/gallery-collections/change-collection-details.png)
